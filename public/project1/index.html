TensorFlow.js


A library for machine learning in JavaScript

This is a library where you can develop Machine Learning models in JavaScript and use it directly in the browser. The library is built on top of the TensorFlow which is based on C++. It is developed by the Google Brain team.

How it works

There are three major ways you can use this library

  • Run existing models
  • Retrain existing models
  • Develop ML with JavaScript

This site will include a page with links to demos and how to get started.
