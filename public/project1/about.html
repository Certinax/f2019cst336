TensorFlow


Developed in 2011

TensorFlow started out as a project at google back in 2012. It was developed for their machine learning / deep mind library. After a 3 years they open-sourced the code under Apache License.

History

Initially TensorFlow was only available for JavaScript through an API where the machine learning model was enabled on a server. A client had to send JavaScript request in order to get results.

Couple of years later, in 2017, Deeplearn.js was released which had the goal to achieve the machine learning models without using an API. Due to performance issues and that JavaScript cannot be run on GPU, WebGL was introduced. This is a browser version og OpenGL which enable the possibility to run JavaScript code on the GPU.

In 2018 Deeplearn.js got merged into Google's TensorFlow project.
