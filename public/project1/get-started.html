Getting started

How to add TensorFlow.js to your project


Add with an import in head

                        
<script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@0.12.0"><script>

Or use npm/yarn

                        
npm install @tensorflow/tfjs
                    

                        
yarn add @tensorflow/tfjs

TensorFlow - What is a tensor?


A tensor is basically a generalization of vectors and matrices to higher dimensions. They can be viewed at as multidimensional array where you can additional information and data.


Example on how to create a tensor from a flat array

                        
const shape = [2,2];
const b = tf.tensor([1, 2, 3, 4], shape);
console.log('shape:', b.shape);
b.print();
